## 最终成功方式

我将资料库初始化功能整合到 main.py 中，
所以只要执行：

uv run main.py

系统就会自动检查 database 资料夹和 restaurant.db，
如果资料表不存在，会自动建立 restaurants 表格，
如果没有资料，会自动加入预设餐厅资料。

这样老师或助教在执行时，不需要额外执行 init_db.py，
可以减少 no such table: restaurants 的错误。
